== Description ==

Baselayer is the lightweight foundation beneath the versatile Powder theme. Built for clarity and speed, it reflects thoughtful simplicity and natively supports the block editor. Whether creating for yourself or clients, Baselayer delivers the essentials: fresh, clean, and ready to go.
